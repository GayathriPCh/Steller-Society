# Steller-Society
Music player with MERN stack

cd Basic_music_player
npm install  
npm install express axios cors
npm install --save-dev nodemon
npm install react-router-dom
npm i react-icons 
npm install react-chat-engine --save --force
npm start 

In a new terminal
cd Server 
npm init
npm install express axios cors
npm install --save-dev nodemon
npm run start
